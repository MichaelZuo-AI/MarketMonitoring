## Getting Started

### Prerequisites
- Node.js & npm
- Python 3.9+
- A Google Cloud API Key with access to Generative AI models.

### Installation

1.  **Clone the repository**
    ```bash
    git clone <repository-url>
    cd world-news-summarizer
    ```

2.  **Setup Backend**
    ```bash
    cd backend
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```
    *Create a `.env` file in `backend/` and add:* `GOOGLE_API_KEY=your_api_key_here`

3.  **Setup Frontend**
    ```bash
    cd ../frontend
    npm install
    ```

### Running the App

1.  **Start the Backend Server**
    ```bash
    # In backend/ directory
    uvicorn main:app --reload --host 0.0.0.0 --port 8001
    ```

2.  **Start the Frontend Client**
    ```bash
    # In frontend/ directory
    npm run dev
    ```

3.  **Access the App**
    Open **http://localhost:5173** to browse and summarize news!
